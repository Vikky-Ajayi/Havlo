# HAVLO — Property Platform Frontend

A complete React + TypeScript frontend built from the HAVLO Figma design.

## Tech Stack
- **React 18** + **TypeScript**
- **Vite** (build tool)
- **React Router v6** (client-side routing)
- **Google Fonts** — Syne + Bebas Neue + DM Sans

## Getting Started

```bash
npm install
npm run dev        # → http://localhost:5173
npm run build      # production build
npm run preview    # preview prod build
```
